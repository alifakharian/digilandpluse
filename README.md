# وب‌سایت فروشگاهی (React + Tailwind)

این پروژه یک وب‌سایت فروشگاهی ساده و قدرتمند است که با استفاده از **React** و **Tailwind CSS** ساخته شده است. این پروژه توانایی من را در ساخت وب‌اپلیکیشن‌های مدرن و واکنش‌گرا از ابتدا بدون استفاده از پکیج‌های آماده برای ویژگی‌های اصلی نشان می‌دهد.

## 🎨 ویژگی‌ها
- **تم شب و روز**:
  - کاربران می‌توانند بین تم شب و روز سوئیچ کنند.
  - تم انتخاب‌شده (شب یا روز) در **LocalStorage** ذخیره می‌شود تا در جلسه‌های بعدی نیز قابل استفاده باشد.
  - **لوگوی سایت** به صورت داینامیک بر اساس تم تغییر می‌کند.

- **سبد خرید**:
  - کاربران می‌توانند محصولات را به سبد خرید خود اضافه کنند.
  - اطلاعات سبد خرید در **LocalStorage** ذخیره می‌شود تا حتی پس از بارگذاری مجدد صفحه، محتوای سبد حفظ شود.

- **ورود و ثبت‌نام**:
  - کاربران می‌توانند با اطلاعات خود وارد سیستم شوند.
  - پس از ورود، یک پنل شخصی برای کاربر ساخته می‌شود که می‌تواند اطلاعات پروفایل و تنظیمات خود را مشاهده کند.

- **اطلاعات محصولات**:
  - جزئیات محصولات از طریق **API** دریافت می‌شود و در صفحات مختلف نمایش داده می‌شود.
  - سایت کاملاً واکنش‌گرا است و تجربه کاربری راحتی را در دستگاه‌های مختلف فراهم می‌کند.

## 💻 تکنولوژی‌های استفاده‌شده
- **React.js**: برای ساخت رابط کاربری و مدیریت وضعیت اپلیکیشن.
- **Tailwind CSS**: برای استایل‌دهی سریع و کارآمد رابط کاربری.
- **LocalStorage**: برای ذخیره تنظیمات تم و اطلاعات سبد خرید.
- **API Integration**: برای دریافت اطلاعات محصولات و نمایش جزئیات آن‌ها.
