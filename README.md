# فروشگاه اینترنتی (React + Tailwind)

این پروژه یک سایت فروشگاهی ساده و قدرتمند است که با استفاده از **React** و **Tailwind CSS** طراحی و توسعه داده شده است. هدف این پروژه، نمایش توانایی‌های من در پیاده‌سازی یک سایت مدرن، بهینه، و کاربردی بدون استفاده از پکیج‌های آماده برای قابلیت‌های کلیدی بوده است.

## 🎨 ویژگی‌ها

- **تم شب و روز**:
  - امکان تغییر تم به شب یا روز.
  - وضعیت تم (شب/روز) در **LocalStorage** ذخیره می‌شود تا در بازدیدهای بعدی کاربر حفظ شود.
  - **لوگو سایت** به‌صورت پویا با تغییر تم عوض می‌شود.
  
- **سبد خرید**:
  - امکان افزودن و مدیریت محصولات در سبد خرید.
  - داده‌های سبد خرید نیز در **LocalStorage** ذخیره می‌شود.
  
- **ورود و ثبت‌نام**:
  - صفحه ورود با امکان وارد کردن اطلاعات کاربر.
  - پس از ثبت‌نام، یک پنل شخصی برای کاربر ساخته می‌شود.
  - اطلاعات پنل شخصی تا زمانی که کاربر از حساب خود خارج نشود باقی می‌ماند.

- **نمایش اطلاعات محصولات**:
  - محصولات و جزئیات آن‌ها از طریق **API** خوانده و در صفحات مختلف نمایش داده می‌شود.
  - طراحی واکنش‌گرا (Responsive) و زیبا برای نمایش اطلاعات محصولات.

## 💻 تکنولوژی‌ها و ابزارها

- **React.js**: برای ساختاردهی و مدیریت رابط کاربری.
- **Tailwind CSS**: برای طراحی سریع و بهینه رابط کاربری.
- **LocalStorage**: برای ذخیره‌سازی وضعیت تم و سبد خرید.
- **API Integration**: برای دریافت اطلاعات محصولات و جزئیات.
